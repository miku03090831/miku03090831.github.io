PAT 1147 & 1155 Heaps & Heap Paths
发布于 2019-11-15   更新于 2019-11-15 data structures and algorithms 热度 0 喜欢 0

上次发的关于堆的题有些简单了。这次来点稍微有点意思的题目，这两道题目很是相似，不过倒是没有用到什么堆的特别操作，只是用了堆（数组）的形式去储存数据，用dfs（深度优先搜索）算法，去遍历堆，然后判断这是最大堆还是最小堆。

1155 Heap Paths

题目

Your job is to check every path in a given complete binary tree, in order to tell if it is a heap or not.

Input Specification:

Each input file contains one test case. For each case, the first line gives a positive integer N (1<N≤1,000), the number of keys in the tree. Then the next line contains N distinct integer keys (all in the range of int), which gives the level order traversal sequence of a complete binary tree.

Output Specification:

For each given tree, first print all the paths from the root to the leaves. Each path occupies a line, with all the numbers separated by a space, and no extra space at the beginning or the end of the line. The paths must be printed in the following order: for each node in the tree, all the paths in its right subtree must be printed before those in its left subtree.

Finally print in a line Max Heap if it is a max heap, or Min Heap for a min heap, or Not Heap if it is not a heap at all.

题目分析

其实还挺简单的一道题，就如题目所说，题目的输入是完全二叉树的层序遍历，层序遍历和用数组表示完全二叉树是等同的。所以接下来的任务就是走遍完全二叉树的每一条path（当然是借助数组实现），然后看看是不是符合最大/最小堆的要求（父节点永远比子节点大/小）。

所以问题的核心就在于解决，如何按照path去遍历树。这里我们用的是深度优先搜索的方法（也叫回溯法），即沿着一条路走到尽头，然后回到前一步的岔路口，并且走另一条路。

1147 Heaps

题目

Your job is to tell if a given complete binary tree is a heap.

Input Specification:

Each input file contains one test case. For each case, the first line gives two positive integers: M (≤ 100), the number of trees to be tested; and N (1 < N ≤ 1,000), the number of keys in each tree, respectively. Then M lines follow, each contains N distinct integer keys (all in the range of int), which gives the level order traversal sequence of a complete binary tree.

Output Specification:

For each given tree, print in a line Max Heap if it is a max heap, or Min Heap for a min heap, or Not Heap if it is not a heap at all. Then in the next line print the tree’s postorder traversal sequence. All the numbers are separated by a space, and there must no extra space at the beginning or the end of the line.
