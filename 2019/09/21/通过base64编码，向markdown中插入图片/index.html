一. base64编码的原理

这一段讲的是原理，不懂的话问题也不大。如果不想看的话可以直接跳到下一段。只是这一段会让你明白何为base64，而不仅仅是如何使用它。

我们都知道图片是二进制文件，而base64就是一种把二进制转换成字符串的编码方式。首先我们要准备一个类似于“字典”的东西，0-63的数字分别对应一个不同的字符。

然后我们拿出二进制文件中的三个字节（byte），一个字节有八位（bit），这样就有了24位0和1组成的位串。我们把这24位分成四组，每一组有6位，并且每一位上都是0或1，这样的话，把每一组表示的二进制数（6位0或1）换成十进制，则会得到一个0-63的整数，再根据前面的“字典”，我们就把这一组的六位换成了一个字符。前面说分成四组，那么我们就会得到四个字符。

到这里，我们把二进制文件中的三个字节换成了四个字符，然后我们三个三个的去处理二进制文件的字节，如果到最后剩的字节数不够3，就在末尾补0凑够三个字节（补了多少个字节也会记下来，否则解码时就有歧义了）。

执行完上面的操作，我们就把一个二进制的文件（不仅限于图片），编码成了字符串（我觉得原理还是挺简单的）
