此花亭日记（一）
发布于 2019-07-28   更新于 2019-08-16 动漫 治愈 热度 0 喜欢 0

想了想，关于此花亭有太多想说的了，放在一篇文章里怕有点长，所以就写一个系列吧，来记录此花亭里面那些触动心灵的时刻

此花亭的那些小狐狸们

    真的是看了此花亭，我就喜欢上了狐狸（笑）。此花亭的每个人，都是独一无二的，个性鲜明，各有各的萌点。但每个人也都有一个共同的特点，就是有一颗善良的心。这也是此花亭的主题吧，善良的心，温柔待人。此花亭里，有天真温柔的柚酱，也有傲娇有刻苦的皋，还有美丽又少女心萌动的莲，直爽开朗的枣，可爱可爱可爱的樱，还有情商高到爆炸的大姐姐——女侍长桐。当然还有亲爱的老板娘啦，老板娘化完妆也超漂亮的！

从给自己放个假开始

    也想了很久，关于此花亭到底该怎么写。也想过按照集数来写，但是那样看上去终究显得有些不自在，感觉像是在写什么观后感之类的。不想为了写而写，而是纯粹的有所感悟，而记录下此时的感受，才是我写随笔的初心。

    柚初来此花亭的时候，是由皋负责带的。皋是一个对工作很认真的完美主义者，责任感也很强。对工作负责总归是好事，不过有的时候绷得太紧了，结果也可能不如所愿。

    第一话里面有位药店老板来此花亭休养，也许是看到了皋对自己的要求太严格了，就跟皋和柚开了个小玩笑，说自己的药材不够了，需要几颗星星。皋一脸懵逼，这你跟我开玩笑呢吗，但是柚却接下了委托。天真的柚真的很可爱呢，对着夜空扔石头，想要把星星打下来。正好一颗流星划过，柚告诉客人有颗星星掉在山的那一边了。认真又严肃的皋说那是客人胡言乱语，客人笑了笑，说这不是胡言乱语，这是玩耍之心。无论工作多么严肃，责任感多么强的人，都要记得给自己放个假。客人还送了两人糖果作为谢礼，说这是能解开心结的药。

    是的呢，有些时候我们可能也会很忙，忙的过头，忙的忘了自己为什么而忙，囿于心结，活的不像样子。这个时候，就应该给自己放个假，重启一下，解决百分之90的问题哈哈哈。

留意身边的美

    第二话里面也有讲到类似的话题，桐要让皋放一天的假休息一下。皋和柚泡温泉的时候对着月亮品酒聊天。我也很羡慕这种生活啊，翻日历的时候看到明天四个大字“今日无事”，心里就很放松，可惜在学校没什么机会喝着清酒泡温泉哈哈哈。盯着月亮的皋，也不禁开始思考，自己究竟多久没有像这样仰望星空了。柚突然说了一句话，“世间万物就这样和我们毫无关联的独自变化”，突然让我有点想起了苏轼的《赤壁赋》。“盖将自其变者而观之，则天地曾不能以一瞬。自其不变者而观之，则物与我皆无尽也，而又何羡乎！”是啊，在你埋头工作，亦或是努力学习的时候，世界还在一刻不停地运转着。而又如柚接下来所说，“一眨眼，一低头，景色立刻就会不一样。没有察觉到这个的话，就会感觉很可惜呢”

    看到这里我真的感慨万千，是啊，平日里太忙了，忙得没有时间去仔细品味身边的美景。景色总是照片里的好看，因为亲眼所见的时候总顾不上去仔细欣赏，只有看别人拍的照片的时候，才能静下心来去细细品味。当初看这部番的时候我正是高三的上学期，十月番播出的时候我应该是刚考完第一次月考，考得不错，是我当时的高中最好成绩，年组排18还是19来着。

    不过虽然考得好，我压力也很大，因为到了高三大家都开始努力学习，虽然班级氛围整体没那么压抑，但是我心里总感觉很累。看了这个我立刻就想起来了赤壁赋，又仔细地去回味了一下，发现自己确实错过了很多很多。每天学习很累，也不开心，错过了多少美丽的事物啊。在上体育课的时候（没错高三还有体育课，高考前一个礼拜都有体育课），从高三楼到操场的路上，我也试过一个人慢慢地走，抬头看天空，初秋的实验园，天气凉爽，景也很美（过几天看看能不能回去拍几张照，哈哈不过只能拍暑假或者寒假的实验园啦）。也大概是从那时起养成的习惯吧，晚自习的课间会去操场散个步。二十分钟的课间，从高三楼走到操场，绕着走个一两圈，再回去，刚好。散步的时候就放空自己，不去想晚自习没做出来的数学题，不去想老师留了多少作业，只去欣赏夜晚的实验园。有年轻的小情侣们也在散步，有同学们在踢足球，也有夜跑的同学不断从我身边超过去。真好啊，想着想着就感觉好豁达，自己对于世界来说渺小的很。即便自己困扰于书上的一道习题，世界也不会停下脚步等你，它依旧不停的向前走。即便你再怎么生气，烦恼，它也不会停下来。所以，也没必要去烦恼什么，不如停下来，仔细欣赏一下身边的美。有时候我们活得太累了，已经出发了太久，该让自己休息一会。背负的担子太沉重了就放下来一点，轻装前行，能走得更远。

接下来的路

    这学期的期末出分的时候我也很伤心啊，微积分明明我学的很认真的，考试也感觉考的还不错，结果期末才答了67。可恶啊，期中感觉没答好都还有91分。结果微积分喜提形策出分前的最低绩点。当时真的是很难受了，妈妈也是来安慰我，还有点怕我想不开。而没心没肺的我难过了一阵，打了几把游戏之后也想开了。绩点这东西本来就很随缘的，付出和回报本来就不一定成正比。就像认真学了一学期的微积分，考砸了，而复习了一天的大物考满了。但是成绩对我真的那么重要吗，我仔细想了下，成绩对我都有什么用。好像除了评奖保研，也没什么用了。最后找工作，再漂亮的成绩单也比不上几个拿的出手的大项目，一行行规范的代码。是的，成绩会骗人，但是实力不会。真正学到了多少东西，自己心里最清楚。学到了东西，对将来的自己能有所帮助，提升自己的水平，就是好的。成绩什么的，偶尔一次考砸了，以后注意一点，不会影响到将来就行了。

    长久纠结于这种事情，也没什么意义，如上面所说的，自己实在是太渺小了。在自己没什么实力的时候，想凭自己去撼动世界，甚至连改变别人都是件很困难的事情。所以，我也学会了，缩小自己的圈子，专注于更少的方面。走的时候全力以赴，累了就停下来看看风景。

共勉
