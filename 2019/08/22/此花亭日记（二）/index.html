此花亭日记（二）
发布于 2019-08-22   更新于 2019-08-22 动漫 治愈 热度 0 喜欢 0

啊学长组的工作终于做完了。新生们开始军训了，而我在看动漫，写写随笔。时不时去操场上看一眼新生们军训的样子，就觉得自己现在的生活是多么的幸福。没错，为什么我不反感军训甚至很支持？别问，问就是反正今年不是我军训（误）。

昨天晚上带新生出去吃烧烤，回到寝室已经快一点了。然后早上一睁眼睛，十一点了。快乐，久违的自然醒。

今天隔了几天又打开了此花亭，挑了一集看。是讲去看祭典的那一集呢，哈哈哈看到柚酱对祭典的印象，我就想到了逛漫展的我。小狐狸还是很可爱的呀，天真的样子，每次一看到都感觉心里的伤都被治愈了。有的时候真的是走着走着就感觉心里面累计了太多的垃圾，每隔一段时间就要清一下心里的缓存，给自己放个假。

好吧最近确实压力比较大，在细说今日份的小狐狸之前，我还是想适当发泄一下压力。压力主要就来源于身边的人吧，身边的人太过于优秀了。而我又很容易受身边的人影响，本来对自己感觉还算可以，但是知道了身边的人有多优秀后，才知道自己还差得远呢。给新生们开班会的时候，下一届的班主任说了一句让我印象很深的话，得志与民由之，不得志独行其道。语出《孟子》，班主任重点强调了后半句。他说，你要容忍身边的人比你优秀。或者换个我理解的意思，就是你要接受这个事实，身边优秀的人确实很多。其实我也从上了高中就发现了，我不是永远顶尖。即便是你很努力了，因为先天基础的差距，再加上努力的程度也可能比不过别人等等原因，别人就是比你优秀。我一遍又一遍的告诉自己，可是自己又总觉得有点遗憾。然后就这样下去压力就有点大吧，不过也还好啦，新的学期要开始了。大一一年吧，成绩也还说得过去。不过可能因为学分有点少，排名会吃点亏。新的学期还是要重新开始，把过往的一切，高中，大一都忘掉，留下的只有努力的方向就够了。

说到祭典呢，我有机会也想去霓虹国体验一下。主要是，有点喜欢日本的一些神明啊，妖怪之类的传说。倒不是出于什么猎奇的原因，而是单纯的觉得这些传说之类的，虽然讲的都是神啊妖的事，但是却很有人情味。可能是看过夏目友人帐的原因吧，我觉得神啊，甚至包括部分的妖啊，如果真的存在的话一定也是很温柔的。那些传说里面，或许也有着哀思，有着离愁，但是也有着温暖人心的美好。

而且，还有什么篝火舞会之类的也会很有趣吧（下图是小狐狸在学舞）

不过本集的重点还在后面哦，后面莲和枣一起去祭典的时候，真的是甜到爆炸了。不得不说莲真的是少女心爆棚啊，为了自己喜欢的人精心打扮，又愿意迁就喜欢的人。心思也非常细腻，忍着心里的失落与寂寞，想留下一点夏日的美好回忆。而枣也没有那么钢铁直男（女）啦，她还记得上次去祭典的时候莲就是因为脚疼而先回去了。所以这次她也特别留心了一下，并且主动提出背着或者抱着莲。枣虽然平时总是大大咧咧的，还像个男孩子一样，但确实也是很会照顾人的。啊呀莲的小心思瞬间就被看穿了，害羞的样子也是很可爱呢。然后终于等到了祭典的正片，放烟花啦，以后我去日本的时候一定也要看一次烟火大会！如果能和喜欢的人一起去就更好了。

随着“啪”的一声烟花炸开，夏日的回忆永远刻在了心底。即便过去了很多年，这份回忆也不会褪色。

回想起来，确实会有和朋友们一起出去玩，玩得很开心的时候呢。很庆幸能有很多朋友，非常会照顾人，总感觉和他们一起出去不用顾虑什么，只顾着开心享受就完了。辛苦的时候去回想一下那个时候开心的事，也就能坚持下去了。

总之新的学期就要开始了，这个学期还是要多学点东西，然后加倍努力！
