python的print函数
发布于 2019-08-10   更新于 2019-08-11 python 热度 0 喜欢 0

今天要写一个倒计时的东西，就是在终端打印出一行字，显示倒计时还剩多久，一秒一刷新的那种。我首先想到的是，隐约记得print函数有个end参数来着。然后就去查了一下实现的方法，由此也引发了一些思考，不仅限于print函数，还包括一些转义字符，字符串等等的东西，我都去查了一下。

虽然还有些东西我也没有完全弄懂，不过还是有些收获的，我相信一些新手们看了这篇文章也会有所收获，就决定先写出来，慢慢再补充

print函数的参数

这个print(*objects, sep=' ', end='\n', file=sys.stdout)是print函数原型

objects就是要输出的对象，是可变参数（即可以有多个，用逗号隔开）

sep就是输出的间隔。当objects有多个时，objects两两之间用sep参数接收的符号隔开，默认为空格

end是输出结束后打印的字符，默认为’\n’换行符

file则是输出的目标，默认的sys.stdout就是控制台，我们通常的print都是把结果打印到控制台上

我们主要来说说后面两个参数

end参数

默认为’\n’，我们如果不想让他换行，则可以写end=””,两个引号中间什么也不加

而如果要实现我在一开头所说的，想要实现打印倒计时的效果，则需要在一行内反复输出，

我们就需要把end设为’\r’，’\r‘的意思是回车（把光标移到当前行的行首）

我们现在讲的回车键，摁下后是换行并把光标移到行首。这个其实是’\n‘的效果，不是’\r’哦

真正的回车，是来源于打字机。在打字的时候，车（carriage,就是会左右移动，纸固定在这上面，然后移动车，就可以在纸的不同位置打印）会不断向一个方向移动，当打完一行的时候，我们需要把车移回来，这个动作叫回车。

小结

本质上，我们用print()或者write方法，都是要将对象以文本形式，输出（或者说写入）到某个地方，比如一个文件，或者一个file-like object(总之能读写的对象)，或者说stdout流。这篇文章一开始是打算记一下自己关于print函数的发现，可写着写着就有些偏题，写到后面就是关于文件IO的东西了。这里再次推荐廖雪峰老师的博客，关于文件IO部分的讲解。
