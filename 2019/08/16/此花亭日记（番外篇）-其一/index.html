此花亭日记（番外篇） 其一
发布于 2019-08-16   更新于 2019-08-16 动漫 治愈 热度 0 喜欢 0

    这篇说是此花亭的番外，其实更多的是关于佛教中的一些传说。缘由是复习到第六集的时候，有一个尚未投胎的鬼魂欺骗了柚，想要拉着柚一起渡河投胎，心灵善良的柚看穿了她的身份，却依旧陪伴着她。最后鬼魂良心发现，独自渡过了河。而片尾说那条河是三途河，我就好奇去查了一下三途河。

    三途河，是佛教传说中冥界的河。普遍认为三途河和忘川是同一条河。插一句，其实本来我对于佛教的东西并不感兴趣，对于什么生死轮回的东西，我虽然相信着科学，但也保持着敬畏。因为有些东西吧，很难解释，也很难分出来绝对的对错，所以有的时候感觉更像是，信则灵，不信则不灵的感觉。

    但是说实话，我还是觉得这些传说或者是怪谈之类的还是挺有意思的。至少，能给我一点心灵的触动，让我时刻铭记着如何更好地活下去。

注明：本文参考资料来自于百度百科三途河条目。另，由于本人对此方面并不了解，因此切勿较真，也希望点进来的朋友们能得到一点启示。

三途河，又名忘川河。之所以名叫三途，是因为河水有三档流速，根据渡河的人，前世罪孽轻重，渡过不同流速的河。相传河旁有一老妇脱下来者的衣服，挂在树枝上。大概是衣服象征着前世的罪孽吧，因罪孽的轻重，而树枝垂下的高度也不一，阴间便也会依此来给你定罪。

有些时候也会不禁问自己，我现在的罪孽有几分重呢？我那个时候是不是不留下来比较好？等等。其实，人最多也只能做到问心无愧，剩下的听天由命。不必太过勉强自己，保持一颗善良的心，别给自己太大的压力，压力太大了心灵也会变形。

既然都说到冥界阴间之类的东西了，那就不妨多说点。之前在看别的动漫的时候也看到过曼珠沙华，即红色彼岸花。花语大概就是生死两隔的意思吧，总之在黄泉路上两侧长满了红色彼岸花，那是一片血红的颜色。路的尽头就是忘川了。忘川河水呈血黄色，里面尽是不得投胎的孤魂野鬼。而与我在动漫中看到的不同的是，渡过忘川不是必须乘船。有一座横跨忘川的桥，这个桥的名字想必大家都很熟悉，奈何桥。之所以叫奈何桥，是因为此时鬼魂还保留着前世的记忆，记得自己未了的愿望，却也只能空叹一声，故名奈何桥。

奈何桥的尽头，是一个叫望乡台的地方。鬼魂在那里可以最后望一眼你此生的爱恨情仇，你的魂牵梦绕，你今生最爱的那个TA，你来世想等的人。。。哦对，忘说了，走上奈何桥之前，会看到桥边有一块石头，叫三生石，能照出你前世的模样。你的生死轮回，都被刻在了上面。据说也可以把此生最爱的人的名字刻在上面，只是下辈子能否再见，谁又说得准呢？

过了望乡台，就要忘却最后的执念了。人在转世投胎的时候，不能带着前世的记忆。没错，是时候喝下孟婆汤了。

孟婆汤，也叫忘情水，由忘川的水熬成。一生的纠葛，都会随着一碗下肚而忘得干干净净，牵挂之人，痛恨之人，统统忘记。而有的人若是爱得太深，不愿喝孟婆汤，便必须跳入忘川水中，受尽千年折磨，才能投胎。这千年里，还要看着自己所爱之人一次次过桥。等到千年过后，可能所爱之人，已非所爱。

当时读到这里我也是感慨良多，因为我心里也有放不下的人。刚上大一的时候，我以为自己放下了，一个学期都没怎么跟她聊过。但是等到假期一见面，我知道我还是忘不了她。然后，临开学前，我和她聊的更是频繁，一直到开学一个多月，我们基本都在保持每天聊天，我这个时候就已经意识到，我已经陷得很深，但是却不想着挣脱。结果自然，每个人都有自己的忙碌，谁也不会天天去等着你的消息然后回复，我们聊得没那么频繁了，我开始有点心烦意乱。本来大一下开学我的学习状态还是蛮好的，结果后来慢慢的也是有点烦心 ，学着学着就经常感觉很乱。不过还好还是挺住了吧。然后中间有一段我还想着，“你对我这么冷淡，那我也干脆不理你好了”。然后过了好长时间（实际上是我感觉过了很久），终于忍不住又发了条消息。结果一看聊天记录，距离上一条消息仅过了十天。什么嘛，我感觉过了一个月都有了的。。。

忘不掉啊，还是忘不掉

好了，这次写的感觉有点乱，因为本身写的东西就比较奇怪吧，一般没人会在随笔里面写这种东西吧哈哈哈。

那么，你在喝下孟婆汤之前的一刻，脑海里回想的是谁的脸呢
